# Export SQL Data to Excel

ExportToExcel is windows desktop application to export SQL data to Excel file.

## Intro
This desktop application enables users to securely connect to a SQL Server instance by providing the server name and authentication credentials. Once the connection is successfully established, the application automatically retrieves and displays all available databases in a dropdown list.

After selecting a database, the application loads and presents all associated tables in a second dropdown. Users can then choose a specific table and export its data to an Excel file. Before the export process begins, the application prompts the user to select a destination folder where the generated Excel file will be saved.

The application provides a simple and intuitive workflow for browsing SQL Server databases and exporting table data to Excel efficiently.
